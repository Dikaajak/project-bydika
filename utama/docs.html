DOKUMENTASI API

PENGANTAR

Selamat datang di API DIKA PEDIA. API ini memungkinkan pengembang untuk mengintegrasikan layanan penyedia nomor (OTP) kami ke dalam aplikasi mereka sendiri secara otomatis.

Penting: Jangan pernah membagikan API Key Anda kepada siapapun. API Key digunakan untuk autentikasi dan memotong saldo akun Anda saat melakukan transaksi.

AUTENTIKASI

Gunakan Header berikut pada setiap permintaan API Anda:

x-api-key: YOUR_API_KEY_HERE

1. CEK PROFIL & SALDO

GET https://api.domain.com/v1/profile

Parameter Tipe Keterangan
api_key String Wajib diisi

Contoh Response Sukses:

{ "status": true, "data": { "nama": "Dika", "saldo": 50000, "role": "MEMBER" } }

2. AMBIL NOMOR (ORDER)

POST https://api.domain.com/v1/order

Parameter Tipe Keterangan
service_code String Kode aplikasi (ex: wa, tg)
country_id Int ID Negara

Contoh Response Sukses:

{ "status": true, "order_id": "99281", "number": "6281818934870", "price": 2500 }

DAFTAR ERROR

Kode Arti
INVALID_KEY API Key salah atau tidak ditemukan.
LOW_BALANCE Saldo akun tidak mencukupi.
SERVICE_OFF Layanan sedang gangguan/maintenance.
